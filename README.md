# VocabTester

CET-4 单词测试 — 一款原生 macOS 应用，帮助你高效背诵大学英语四级词汇。

## 功能

- **四级词库** — 内置 4414 个 CET-4 词汇，覆盖完整四级考纲
- **看中文写英文** — 根据中文释义拼写对应英文单词，强化记忆
- **灵活题量** — 支持 20 / 50 / 100 题或全部词库随机测试
- **错题本** — 答错的单词自动进入错题库，可针对性复习
- **错题淘汰** — 错题模式下连续答对 2 次自动移出错题库
- **成绩回顾** — 测试完成后展示正确率和错误词汇列表
- **键盘操作** — 全程支持键盘快捷键，流畅答题体验

## 系统要求

- macOS 14.0+
- Apple Silicon (arm64)

## 安装与运行

1. 用 Xcode 打开 `VocabTester.xcodeproj`
2. 确保 `vocab.json` 已添加到 Target 的 Copy Bundle Resources
3. 选择 **Product > Run** (⌘R)

## 项目结构

```
VocabTester/
├── VocabTester_CET4.swift   # 全部代码（数据模型 / ViewModel / UI）
├── vocab.json               # 四级词汇数据（4414 条）
├── AppIcon.icns             # 应用图标
└── Info.plist               # 应用配置
```

## 使用说明

| 操作 | 方式 |
|------|------|
| 提交答案 | 输入后按 ↩ |
| 进入下一题 | 答错后按 ↩ 继续 |
| 退出测试 | 按 Esc 或点击 ✕ |

- 答对后自动跳转下一题（0.9 秒延迟）
- 答错显示正确答案，手动按 ↩ 继续
- 错题本数据持久化保存在本地，重启应用不丢失
